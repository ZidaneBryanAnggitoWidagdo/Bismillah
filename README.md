## Quiz App Sederhana
Project ini merupakan program berbasis Python yang bertujuan untuk 
menyediakan sistem kuis/bank soal interaktif dengan beberapa kategori 
dan level kesulitan. Program ini memungkinkan pengguna untuk menjawab 
soal pilihan ganda, mendapatkan skor, serta melihat hasil akhir setelah 
kuis selesai.

Project ini dibuat sebagai latihan logika pemrograman, pengolahan data, 
serta manajemen file menggunakan Python.

---

## Fitur Utama

- Sistem soal pilihan ganda
- Terdapat beberapa kategori mata pelajaran
- Level kesulitan berbeda
- Sistem skor otomatis
- Penyimpanan data ke file
- Validasi input pengguna
- Ringkasan hasil setelah kuis selesai

---

## Cara Menjalankan Program

1. Pastikan Python sudah terinstall (minimal Python 3.x)
2. Download atau clone repository ini
3. Buka terminal pada folder project
4. Jalankan perintah berikut:

   python main.py

5. Ikuti instruksi yang muncul di layar
